# ⚽ Análisis Manchester City vs Napoli - Champions League

Análisis táctico y estadístico del partido de Champions League utilizando Python y visualizaciones de datos.

## 📊 Características del Análisis

- 📈 **Análisis de xG (Expected Goals)**: Evolución durante el partido
- 🗺️ **Mapas de calor**: Distribución de oportunidades de gol
- 📋 **Dashboard interactivo**: Comparación de estadísticas
- 🔍 **Análisis táctico**: Impacto de la expulsión en el minuto 21

## 🛠️ Tecnologías Utilizadas

- Python 3.x
- matplotlib
- pandas
- numpy
